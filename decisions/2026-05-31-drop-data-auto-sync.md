# スキル・職務経歴データの自動同期を廃止し手動運用に

- 日付: 2026-05-31
- 対象: `src/lib/data/skills.ts` / `career.ts` のデータ更新方式

## 採用した方針

- スキル・職務経歴のデータは**手動で同期**する（大本シート → `*.ts` → PR → マージ）
- 自動化を前提にした記述（docs の「フェーズ 2 自動更新」「外部 API 連携」、コメント等）を削除し、手動運用に統一

## 却下した代替案と理由

- **データ自動同期**（スケジュール実行で外部 API からデータ取得 → 自動 PR）: 方針変更により廃止
- フェーズ 2 の AI 活用ページ拡充自体は維持（廃止したのはデータ更新の自動化のみ）

## 見直しトリガー

- 運用負荷が増え、自動同期の必要性が再浮上した場合（その際は認証・機密管理と自動マージの安全設計を再検討）
