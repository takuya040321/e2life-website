# e2life-website

個人 Web サイト [e2life.dev](https://e2life.dev)

## 概要

AI で開発プロセス自体を設計するエンジニアの個人 Web サイト。
SE エージェントへの差別化資料 + SNS からの流入先 + 直接案件獲得チャネル。

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router, PPR, Turbopack)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS + shadcn/ui
- **ホスティング**: Vercel
- **テスト**: Vitest + Playwright + Lighthouse CI

## サイト構成

- `/` — LP（TOP）
- `/skills` — SE スキル
- `/career` — 職務経歴
- `/contact` — 問い合わせ
- `/ai/` — AI 活用（フェーズ 2）

## 開発

```bash
npm run dev
```
